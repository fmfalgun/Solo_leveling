## PROJECT SCOPE BREAKDOWN

### Technical Scope

**Protocols Analyzed:**
- IEEE 802.15.4 (PHY/MAC layer)
- Zigbee (Network layer, clustering)
- Bluetooth Low Energy (Link layer, GATT/ATT)
- Optional: Thread, LoRaWAN (extended scope)

**Attack Vectors Covered:**
- Cryptographic weaknesses (static keys, weak KDF, mode misuse)
- Authentication failures (default credentials, missing auth, bypasses)
- Protocol violations (state machine manipulation, DoS)
- Hardware/firmware vulnerabilities (JTAG access, firmware extraction)
- Physical attacks (jamming, eavesdropping, side-channels)
- Application layer (insecure APIs, input validation flaws)

**Hardware Supported:**
- Ubertooth One (BLE sniffer)
- CC2531 USB dongles (Zigbee)
- XBee S3B modules (802.15.4, fuzzing)
- NRF52840 DK (BLE 5.0, custom firmware)
- Raspberry Pi 4 (gateway/aggregation)
- Optional: USRP SDR (signal analysis)

**Deliverables (8-10 Tools):**
1. Multi-device packet sniffer
2. Protocol decoder/dissector
3. Cryptographic analyzer
4. Protocol fuzzer
5. Key extraction toolkit
6. Packet injector/replay tool
7. Session hijacking framework
8. Device impersonator
9. Web-based analysis dashboard
10. Automated reporting engine
